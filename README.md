# AI-Powered Real-Time Credit Card Fraud Detection Platform

## Overview
A real-time fraud detection system that uses Machine Learning, Streaming, and Database technologies to identify suspicious credit card transactions.

## Features
- Fraud Detection using XGBoost
- Risk Score Calculation
- Alert Generation
- Real-Time Transaction Streaming
- PostgreSQL Data Storage
- Redpanda (Kafka-Compatible Messaging)
- Future Streamlit Dashboard

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- PostgreSQL
- SQLAlchemy
- Redpanda
- Docker
- Streamlit
- FastAPI
